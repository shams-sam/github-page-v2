I am Shams, a Data Scientist interested in Machine Learning and Advanced Mathematics.

At Practo, I work on NLP and Computer Vision problems to build intelligent healthcare systems driven by machine learning models. I have built systems from ground up for retrieving clinical concepts from free text leveraging NLP techniques and allocating appropriate ICD-10 code using Dense-LSTM Hybrid DNNs.

Before that, I was a part of Practo Search, working on ranking algorithms and internationalization of search across various countries by building the backend using technologies like elasticsearch, lucene. I also worked on Practo Discovery to build nginx level URL discovery and redirection service.

I am an alumni from NITK Surathkal, India.
